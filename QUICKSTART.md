# 🚀 Lorify - Quick Start (2 Minutes)

## Step 1: Get Free API Key (1 minute)

1. Go to **[console.groq.com](https://console.groq.com)**
2. Click **"Sign Up"** (it's free!)
3. Go to **"API Keys"** section
4. Click **"Create API Key"**
5. **Copy** your API key

## Step 2: Add API Key (30 seconds)

1. Open **`script.js`** in any text editor
2. Find **line 5**:
   ```javascript
   API_KEY: 'YOUR_GROQ_API_KEY_HERE',
   ```
3. Replace `YOUR_GROQ_API_KEY_HERE` with your actual key:
   ```javascript
   API_KEY: 'gsk_abc123xyz...',
   ```
4. **Save** the file

## Step 3: Run! (10 seconds)

1. Open **`index.html`** in your browser
2. Start roasting! 🔥

## Troubleshooting

### "API key not configured" error?
→ Make sure you saved `script.js` after adding your key

### "Network error" message?
→ Check your internet connection and verify your API key is correct

### Nothing happens when I click submit?
→ Open browser console (F12) and check for errors
