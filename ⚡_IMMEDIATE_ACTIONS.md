## 🐛 TROUBLESHOOTING

### Problem: "Please configure your Groq API key"

**Solution:**
1. Open `website/script.js`
2. Check line 5 has your actual key (starts with `gsk_`)
3. Make sure you saved the file
4. Refresh your browser (F5)

---

### Problem: Nothing happens when I click submit

**Solution:**
1. Press F12 to open browser console
2. Look for red error messages
3. Common issues:
   - API key is wrong (check it)
   - No internet connection (connect)
   - Typo in the key (re-copy it)

---

### Problem: "API request failed"

**Solution:**
1. Check your API key is correct
2. Go to https://console.groq.com
3. Verify your key is active
4. Try creating a new key if needed

---

### Problem: Page looks broken

**Solution:**
1. Make sure all 3 files are in same folder:
   - `index.html`
   - `styles.css`
   - `script.js`
2. Try different browser (Chrome recommended)
3. Clear browser cache (Ctrl+Shift+Delete)
